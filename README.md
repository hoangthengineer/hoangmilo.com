# hoangmilo.com — Web Profile v1

Personal profile site for [hoangmilo.com](https://hoangmilo.com). Single-page layout with hero, about, experience, skills, projects, and contact.

## Stack

- [Vite](https://vite.dev/) — dev server and static build
- Vanilla HTML / CSS / JS — no framework overhead

## Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build & deploy

```bash
npm run build
```

Output is in `dist/`. Deploy to any static host (Vercel, Netlify, Cloudflare Pages, GitHub Pages, etc.) and point `hoangmilo.com` DNS to it.

## Customize

Edit `index.html` for copy, links, and sections. Styles live in `src/style.css`. Behavior in `src/main.js`.
